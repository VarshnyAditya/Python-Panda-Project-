#Countries Data Analysis using Pandas

This project performs exploratory data analysis on a global countries dataset using Python libraries such as NumPy and Pandas.

The goal of this project is to practice data manipulation, filtering, and querying using Pandas while extracting meaningful insights from real-world country data.

## 📊 Features / Analysis Performed

- Load and explore dataset using Pandas
- Identify the country with the highest population
- Find the capital city of the most populated country
- Identify the country with the least population
- Find the capital city of the least populated country
- Extract top 5 countries with the highest democracy score
- Analyze regional distribution of countries
- Count how many countries belong to specific regions
- Identify the political leader of the second most populated country
- Find how many countries have unknown political leaders
- Count countries that contain "Republic" in their official name
- Identify the African country with the highest population

## 🛠 Technologies Used

The dataset contains information about countries including:
